## 👩‍🎓 졸업을 도와조

```
  덕성여자대학교 학생들의 졸업을 도와주는 웹 서비스입니다

  졸업을 위한 다양한 정보를 쉽게 얻을 수 있으며 졸업 전시회의 아카이빙, 팀원 모집이나
  정보 공유를 위한 커뮤니티가 있습니다

  '졸업을 도와조'를 통해 졸업 요건을 찾고 프로젝트를 수월하게 준비하세요

```
